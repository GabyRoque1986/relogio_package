# relogio_package

**relógio_package** é um package Flutter com widgets otimizados para **Wear OS**, pensado para telas pequenas e interação rápida.  
O widget principal deste package é o **BatteryRingWidget**, que exibe o nível da bateria do dispositivo como um **anel circular**.

---

## Widget principal: BatteryRingWidget

- Mostra a porcentagem da bateria no centro do anel
- Ideal para smartwatches
- Atualiza automaticamente o nível da bateria
- Pode ser usado dentro de qualquer app Flutter

---

## Instalação

No `pubspec.yaml` do seu projeto Flutter, adicione o package local:

  relogio_package:
